# Lando Norris Premium Personal Website

A luxury, F1-inspired personal website showcasing premium frontend design skills with advanced 3D interactions and scroll animations.

üåê **Live Site:** https://lando-site.vercel.app

## Features

### Design System
- **Clean, modern luxury sports aesthetic**
- White/light gray background with strategic whitespace
- **Neon yellow/lime green (#CCFF00)** as primary accent
- Bold, uppercase sans-serif typography (Inter)
- High contrast black text for readability

### Interactive Elements
- **Three.js 3D helmet** in hero section with mouse interaction
- **Interactive helmet gallery** with 3D rotation on hover
- **GSAP scroll-triggered animations** throughout
- Smooth parallax effects
- Responsive hamburger menu

### Sections
1. **Hero Section** - Full-screen with 3D rotating helmet and bold typography
2. **Navigation** - Fixed header with prominent yellow STORE button
3. **Next Race Module** - Floating bottom-left module with track outline
4. **ON TRACK** - Race results, stats, and championship standings
5. **OFF TRACK** - Lifestyle and brand content
6. **HELMETS HALL OF FAME** - 7 unique helmet designs (2019-2025) with 3D interactions
7. **Partners** - Sponsor grid with hover effects
8. **Social Feed** - Social media integration
9. **Footer** - Clean navigation and social links

## Tech Stack

- **Vite** - Fast build tool and dev server
- **Three.js** - 3D graphics and WebGL rendering
- **GSAP** - Professional-grade scroll animations
- **Vanilla JavaScript** - No framework overhead
- **Modern CSS** - Custom properties, grid, flexbox

## Performance

- **Total bundle size:** ~600KB (optimized)
- **Build time:** 2.64s
- **Lighthouse score:** Optimized for performance
- **Lazy loading:** Images load on demand
- **Code splitting:** Separate chunks for Three.js and GSAP

## Images

All images generated with DALL-E 3 in HD quality:
- Hero portrait (F1 driver in McLaren suit)
- 7 unique helmet designs (one per year 2019-2025)
- Racing action shot
- Lifestyle/off-track imagery

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
